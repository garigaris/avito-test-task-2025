# Система управления объявлениями для модерации Авито

Веб-приложение для модерации объявлений на платформе Авито. Система предоставляет  интерфейс для проверки объявлений пользователей, а также просмотра статистики.

## Технологии

### Основной стек:
- **React 18+** - современная библиотека для построения пользовательских интерфейсов
- **TypeScript** - статическая типизация для повышения надежности кода
- **Vite** - быстрая сборка и разработка
- **React Router DOM** - маршрутизация между страницами
- **Axios** - HTTP-клиент для работы с API
- **React Query (TanStack Query)** - управление серверным состоянием и кэширование

### Дополнительные технологии:
- **Recharts** - простая и гибкая библиотека для построения графиков и диаграмм в React. Она позволяет быстро визуализировать статистику объявлений, поддерживает интерактивность (hover, tooltip, легенды) и легко настраивается под кастомный стиль приложения.

## Функциональность

### Основные возможности:
- **Список объявлений** - просмотр, фильтрация, сортировка и пагинация
- **Детальный просмотр** - полная информация об объявлении и истории модерации
- **Статистика** - аналитика работы модератора с графиками


### Дополнительные функции:
- **Сохранение фильтров после перезагрузки** 
- **Горячие клавиши** - ускорение работы модератора
- **Real-time обновления** - автоматическое обновление данных
- **Тёмная тема** - переключатель тем с сохранением в localStorage

## Установка и запуск

### Предварительные требования:
- Node.js v20 или выше
- npm или yarn

### Локальный запуск:

1. **Клонирование репозитория:**
```bash
git clone https://github.com/garigaris/avito-test-task-2025.git
```
## Запуск frontend'a
Перейти в скачанный репозиторий

2. **Установить зависимости:**
```bash
npm install
```

3. **Запуск сервера разработки:**
```bash
npm run dev
```
4. **Открытие приложения в  браузере:**
   перейти на http://localhost:5173


## Запуск backend'a
5. **Перейти в папку сервера c основной папки:**
```bash
cd server
```
6. **Установить зависимости:**
```bash
npm install
```

7. **Запуск сервера разработки:**
```bash
npm run dev
```
